# Traffic Crashes - Vehicles Dataset Metadata

| Column Name | Description |
| :--- | :--- |
| **CRASH_UNIT_ID** | A unique identifier for each vehicle record. |
| **CRASH_RECORD_ID** | This number can be used to link to the same crash in the Crashes and People datasets. |
| **CRASH_DATE** | Date and time of crash as entered by the reporting officer. |
| **UNIT_NO** | A unique ID for each unit within a specific crash report. |
| **UNIT_TYPE** | The type of unit (motor vehicle, non-motor vehicle). |
| **NUM_PASSENGERS** | Number of passengers in the vehicle. The driver is not included. |
| **VEHICLE_ID** | vehicle_id. |
| **CMRC_VEH_I** | cmrc_veh_i (Indicator for commercial vehicle). |
| **MAKE** | The make (brand) of the vehicle, if relevant. |
| **MODEL** | The model of the vehicle, if relevant. |
| **LIC_PLATE_STATE** | The state issuing the license plate of the vehicle, if relevant. |
| **VEHICLE_YEAR** | The model year of the vehicle, if relevant. |
| **VEHICLE_DEFECT** | Type of vehicle defect (if any). |
| **VEHICLE_TYPE** | The type of vehicle, if relevant. |
| **VEHICLE_USE** | The normal use of the vehicle, if relevant. |
| **TRAVEL_DIRECTION** | The direction in which the unit was traveling prior to the crash. |
| **MANEUVER** | The action the unit was taking prior to the crash. |
| **TOWED_I** | Indicator of whether the vehicle was towed. |
| **FIRE_I** | Indicator of whether the vehicle caught fire. |
| **OCCUPANT_CNT** | The number of people in the unit, as determined by the reporting officer. |
| **EXCEED_SPEED_LIMIT_I** | Indicator of whether the unit was speeding. |
| **TOWED_BY** | Entity that towed the unit, if relevant. |
| **TOWED_TO** | Location to which the unit was towed, if relevant. |
| **AREA_00_I to AREA_99_I** | Damage area indicators on the vehicle. |
| **FIRST_CONTACT_POINT** | Point of first contact on the vehicle during the collision. |
| **CMV_ID** | Commercial Motor Vehicle ID. |
| **USDOT_NO** | US Department of Transportation Number. |
| **CCMC_NO** | CCMC Number. |
| **ILCC_NO** | ILCC Number. |
| **COMMERCIAL_SRC** | Source of commercial vehicle data. |
| **GVWR** | Gross Vehicle Weight Rating. |
| **CARRIER_NAME** | Carrier name. |
| **CARRIER_STATE** | Carrier state. |
| **CARRIER_CITY** | Carrier city. |
| **HAZMAT_PLACARDS_I** | Indicator if HAZMAT placards were present. |
| **HAZMAT_NAME** | Name of the hazardous material. |
| **UN_NO** | UN Number for hazardous materials. |
| **HAZMAT_PRESENT_I** | Indicator if hazardous materials were present. |
| **HAZMAT_REPORT_I** | Indicator if a HAZMAT report was filed. |
| **HAZMAT_REPORT_NO** | HAZMAT report number. |
| **MCS_REPORT_I** | Motor Carrier Safety report indicator. |
| **MCS_REPORT_NO** | Motor Carrier Safety report number. |
| **HAZMAT_VIO_CAUSE_CRASH_I**| Indicator if a HAZMAT violation caused the crash. |
| **MCS_VIO_CAUSE_CRASH_I** | Indicator if an MCS violation caused the crash. |
| **IDOT_PERMIT_NO** | IDOT permit number. |
| **WIDE_LOAD_I** | Indicator if the vehicle was a wide load. |
| **TRAILER1_WIDTH** | Width of the first trailer. |
| **TRAILER2_WIDTH** | Width of the second trailer. |
| **TRAILER1_LENGTH** | Length of the first trailer. |
| **TRAILER2_LENGTH** | Length of the second trailer. |
| **TOTAL_VEHICLE_LENGTH** | Total length of the vehicle/combination. |
| **AXLE_CNT** | Number of axles. |
| **VEHICLE_CONFIG** | Configuration of the vehicle. |
| **CARGO_BODY_TYPE** | Type of cargo body. |
| **LOAD_TYPE** | Type of load being carried. |
| **HAZMAT_OUT_OF_SERVICE_I** | Indicator if placed out of service for HAZMAT violations. |
| **MCS_OUT_OF_SERVICE_I** | Indicator if placed out of service for MCS violations. |
| **HAZMAT_CLASS** | Class of the hazardous material. |
